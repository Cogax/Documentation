若要在QuantConnect中使用OANDA Brokerage FX产品，您必须指定QuantConnect的代号而不是OANDA代号。您必须手动在Initialize()方法中添加CFD数据。指定Market.Oanda以在OANDA历史数据上进行回测。 

// 手动添加初始化方法中所需的代号：
public override void Initialize() {
    AddCfd("AU200AUD", Resolution.Minute, Market.Oanda);
}
// 通过专用事件的处理器来访问数据
public void OnData(TradeBars data) {
    data["AU200AUD"].Close;
}
// 通过整合的时间slice方法处理器来获得数据
public override void OnData(Slice data) {
    data.Bars["AU200AUD"].Close;
}

# 手动添加初始化方法中所需的代号：
def Initialize(self):
    sef.AddCfd("AU200AUD", Resolution.Minute, Market.Oanda)
