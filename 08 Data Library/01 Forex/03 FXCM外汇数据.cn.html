QuantConnect提供FXCM（福汇）的39种货币对，用于回测和实盘交易，数据最早从2007年4月开始。FXCM（福汇）货币的价差低于传统做市商，因为FXCM直接通过多家流通量供应商成交交易，提供有竞争力的低价差。FXCM（福汇）每手收费为固定交易费而非根据价差收费。

要在QuantConnect中使用FXCM的货币对，您必须指定QuantConnect的代号而不是FXCM的代号。数据会根据请求自动传递到您的算法中。您必须在Initialize()方法中添加想请求的数据。 

// 在初始化方法中手动添加您想要的货币对：
public override void Initialize() {
    AddForex("EURUSD", Resolution.Minute);
}
// v2.0技巧 - 通过指定的事件处理程序访问数据：
public void OnData(TradeBars data) {
    data["EURUSD"].Close;
}
// v3.0技巧 - 通过整合时间的Slice方法处理器访问数据：
public override void OnData(Slice data) {
    data.Bars["EURUSD"].Close;
}

# 在初始化方法中手动添加您想要的货币对：
def Initialize(self):
    self.AddForex("EURUSD", Resolution.Minute, Market.FXCM)

# 通过整合时间的Slice方法处理器访问数据：
def OnData(self, data):
    data.Bars["EURUSD"].Close
